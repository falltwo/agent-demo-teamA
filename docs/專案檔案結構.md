# 專案檔案結構

專案根目錄與主要目錄的職責說明，方便快速定位檔案。

---

## 根目錄

| 類型 | 檔案 | 說明 |
|------|------|------|
| **入口** | `streamlit_app.py` | Streamlit 主程式（多對話、上傳灌入、問答、Eval 檢視） |
| | `main.py`、`rag_chat.py` | 其他入口（若使用） |
| **核心** | `agent_router.py` | 總管 Agent：工具路由、合約／RAG／專家／圖表 |
| | `rag_graph.py` | RAG 核心：LangGraph、retrieve、generate、retrieve_only |
| | `rag_common.py` | 共用：chunk、Pinecone/Gemini 初始化、embed |
| | `llm_client.py` | LLM 客戶端（Gemini / Groq） |
| **專家與工具** | `expert_agents.py` | 財報、ESG、資料分析、合約審閱專家 |
| | `company_tools.py` | 財報指標、日期解析、季度計畫 |
| | `firecrawl_tools.py` | Firecrawl API |
| | `echarts_tools.py`、`echarts_mcp_client.py` | 圖表 ECharts |
| | `citecheck_client.py`、`gavel_client.py` | 選用客戶端 |
| **灌入與狀態** | `rag_ingest.py` | 離線灌入 data/ → Pinecone |
| | `sources_registry.py`、`sources_registry.json` | 來源註冊表 |
| | `eval_log.py`、`eval_runs.jsonl` | Eval 運行記錄（可選） |
| **設定** | `.env.example`、`pyproject.toml`、`uv.lock`、`.python-version` | 環境與依賴 |
| | `README.md`、`LICENSE`、`.gitignore` | 說明與版權 |

---

## 目錄

| 目錄 | 說明 |
|------|------|
| **docs/** | 專案文件：競賽對齊、RAG 記憶、痛點分析、專案完整度等；入口 [docs/README.md](README.md) |
| **eval/** | Eval 題集（eval_set.json、eval_set_contract.json）與 run_eval.py；結果寫入 eval/runs/ |
| **assets/** | 前端資源（如 custom.css） |
| **data/** | 預設灌入來源（txt、md、pdf） |
| **tests/** | 單元測試 |
| **tools/** | 工具模組（若有） |
| **.streamlit/** | Streamlit 設定（config.toml） |
| **.cursor/** | Cursor 設定、skills、MCP 等 |

---

## 文件索引

所有說明文件集中在 **docs/**，請從 [docs/README.md](README.md) 進入依主題查找。
